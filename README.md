- **Privacy-First** — All data stored locally on-device using Apple's secure on-device storage. No accounts. No servers. No analytics. No tracking.
- **OTA Templates** — New prompt template packs delivered over-the-air without waiting for App Store updates.

## Frequently Asked Questions

### How do I save a prompt?
Tap the **+** button in the Library tab, choose "Write from Scratch" or "Use Prompt Builder," fill in your prompt details, then tap Save.

### How do I organize prompts into folders?
Go to the Folders tab and create folders with custom names + emoji icons. You can move prompts between folders from the prompt detail screen.

### How do I find new prompts?
Visit the Discover tab to browse 200+ curated prompt templates across 11 categories. Tap "Add to Library" to save any template.

### Can I export my prompts?
Yes. Settings → Export Prompts. Saves your entire library as a JSON file you can share or back up.

### Can I import prompts?
Yes. Settings → Import Prompts. Loads prompts from any JSON file.

### Is my data private?
Absolutely. All your prompts are stored locally on your device using Apple's on-device storage. No servers, no accounts, no tracking.

### Will I get new prompt templates?
Yes. We deliver new templates via over-the-air updates so you get fresh content without waiting for App Store updates.

### How is this different from a Notes app?
Notes apps don't structure prompts into folders, don't support `{{variables}}`, don't have a discovery feed of curated templates, and don't track your AI subscription spend.

### Does it support Claude / Gemini / ChatGPT prompts?
Yes — prompts are AI-agnostic. Save once, use across any LLM.

### Will there be Android / web versions?
Currently iOS-only. Reach out if you'd like other platforms.

### How do I update the variable values quickly?
When you tap a saved prompt that contains `{{placeholders}}`, the app pops up a fill-in form. Type the values once and tap Run.
